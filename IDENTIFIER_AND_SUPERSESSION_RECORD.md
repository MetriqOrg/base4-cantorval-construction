## Cardinal-range consolidation

MF-MATH-2026-06 Version 1.0 contained two cardinal-range certificates. Version
2.0 incorporated the seven-atom certificate that had been circulated in two
duplicate working manuscript files: one labeled MF-MATH-2026-07 and one whose
filename temporarily contained 2026-08 while its internal manuscript identifier
remained MF-MATH-2026-07. Those drafts described the same mathematical result.
They were not separate contributions.

The consolidated canonical paper is **MF-MATH-2026-06, Version 2.1**. The
preserved MF-MATH-2026-07 draft is historical only and must not be cited beside
MF-MATH-2026-06 as an independent result.

## Canonical MF-MATH-2026-08

The identifier MF-MATH-2026-08 is assigned exclusively to **Endpoint Walks
Evaluate the Complete Homogeneous Symmetric Norms of Path Graphs**, Version 1.1.
This is a distinct spectral-graph-theory paper and is not part of the cardinal-
range consolidation.

## Citation rule

Citations, repository releases, DOI deposits, and website listings should use
only the canonical identifiers and versions above. Older versions may be kept
for provenance but must be marked superseded and must not replace the current
files silently.
