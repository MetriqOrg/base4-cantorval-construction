# Repository setup and use

Clone the canonical repository:

```bash
git clone https://github.com/MetriqOrg/PRISM.git
cd PRISM
```

The repository itself has no installation step. Each paper's directory contains its release PDF, cover image, complete source/reproducibility ZIP, README, and checksums. Follow the build and verification instructions inside the corresponding source package.

## Branding

The repository header uses the official full-name lockup:

`Metriq PRISM Laboratory — Program for Research in Intelligent Systems and Methods`

Use the shorter laboratory lockup only where the full-name lockup is impractical because of size or legibility constraints.
