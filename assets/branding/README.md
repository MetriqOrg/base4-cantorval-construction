# PRISM repository branding

The repository README uses the official full-name lockup from the Metriq PRISM Brand Kit Version 1.1.

- `Metriq_PRISM_Full_Name_Lockup_Color_Light.svg` — for light interfaces
- `Metriq_PRISM_Full_Name_Lockup_Color_Dark.svg` — for dark interfaces

Use the full-name lockup whenever space and legibility permit. The shorter “Metriq PRISM Laboratory” lockup is reserved for constrained applications.
